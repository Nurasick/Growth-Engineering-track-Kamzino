Part 3 · The Machine

Weekly competitive intelligence —
two-speed pipeline

Signal Surfaces HN · X · Reddit · YouTube
→
Fast Lane · every 30 min Lightweight poll: HN/Reddit/X last 2h → cascade check → threat check → alert if velocity > threshold
Slow Lane · Monday 8am Full scrape → normalize → classify (keyword + Claude Haiku fallback) → rank → dashboard update
→
4 Alert Types CASCADE · THREAT · CREATOR OPPORTUNITY · FRAMING
Live Dashboard FastAPI + Next.js. Velocity feed, weekly trend, creator watchlist, cascade timeline
LLM classifier

≥60% keyword confidence → keyword label (free). <60% → Claude Haiku API (~$0.001/post). 65.5% disagreement rate on ambiguous posts. Cost for 100 calls: $0.087.

Cascade detector

Jaccard similarity on keyword sets, 6h sliding windows. 48 cascades found from real data. Detects FAST_CASCADE (<6h) vs SUSTAINED threads across platforms.

The strongest acquisition story

Not "our product is powerful" — "people like you switched." Migration stories combine comparison + social proof + identity change.

Show, don't announce

The first public asset must travel without a thread or article. For a visual product: a side-by-side replacement, a transformation that looks impossible, or a visible workflow reduction.

The biggest gap right now

35K TikTok vs 846K Instagram

Higgsfield's product is TikTok content. The platform where their output naturally lives has 24× less reach than their static image platform.

Run it as an operating system

Monitor → Select moment → Package as proof/comparison/migration → Distribute through creators → Read signals → Compound. Not a launch calendar. A repeating loop.
